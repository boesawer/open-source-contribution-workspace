## Good Signals

- The issue is clear and scoped.
- The repository has recent activity.
- Setup instructions are available.
- Tests or manual verification steps are possible.
- No one else is already working on the same fix.
- The change can be reviewed without deep project context.

## Red Flags

- The issue is stale and unanswered.
- Multiple pull requests already attempted the same fix.
- The request is vague or much larger than the label suggests.
- Setup is unclear and the change cannot be verified.
- The issue requires access, secrets, or private project context.
